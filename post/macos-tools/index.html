macOS工具：

包管理器(必备神器)：Homebrew

浏览器：Chrome,Safari

编辑器：MacVim,Visual Studio Code(VS),MacDown,ProperTree,Numbers,Pages,WPS

压缩：The Unarchiver

API工具：Postman

通信：WeChat,QQ,钉钉,Telegram,LINE

密码管理(付费)：1Password

远控：向日葵,ToDesk,Microsoft Remote Desktop

ssh/ftp工具：Termius,FileZilla

梯子：qv2ray(真香),ClashX(用不惯)

数据库：DBeaver,Another Redis Desktop Manager

虚拟机：VirtualBox

U盘刻录：balenaEtcher

渗透(付费/社区版)：Burp Suite Community Edition

ssh工具(IOS)：Termius

付费：Transmit,Navicat Premium,IntelliJ IDEA,GoLand,PhpStorm,PyCharm,Redis Desktop Manager

付费(IOS)：Shadowrocket

Windows工具：

Xshell,Xftp（认准官方网站www.netsarang.com或www.xshell.com，找到免费许可页面下载。）
