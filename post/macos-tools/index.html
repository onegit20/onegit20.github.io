macOS工具：

包管理器(必备神器)：Homebrew

浏览器：Chrome, Safari

编辑器：Visual Studio Code, MacDown, ProperTree

压缩：The Unarchiver

API工具：Postman

密码管理(付费)：1Password

远控：向日葵, ToDesk, Microsoft Remote Desktop

ssh/ftp工具：Termius, FileZilla

梯子：qv2ray

数据库：DBeaver

虚拟机：VirtualBox

U盘刻录：balenaEtcher

渗透(付费/社区版)：Burp Suite Community Edition

ssh工具(IOS)：Termius

本地shell(IOS)：a-Shell, iSH, LibTerm

付费(IOS)：Shadowrocket

Windows工具：

Xshell, Xftp（认准官方网站www.netsarang.com或www.xshell.com, 找到免费许可页面下载。）
