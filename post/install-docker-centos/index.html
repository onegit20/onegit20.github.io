安装docker-ce

此示例操作系统为CentOS 8，如果yum install containerd.io有报错，根据错误提示去阿里云centos镜像站找合适的rpm包安装。

1
2
3
4
5
6
7

sudo yum install -y yum-utils
sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install -y docker-ce docker-ce-cli containerd.io
sudo systemctl start docker
sudo systemctl enable docker

参考链接：

https://docs.docker.com/engine/install/centos/

阿里镜像加速器

XXXXXXXX.mirror.aliyuncs.com替换成自己的加速器地址，地址在下面给出的参考链接里查找

1
2
3
4
5
6
7
8

sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://XXXXXXXX.mirror.aliyuncs.com"]
}
EOF

sudo systemctl daemon-reload
sudo systemctl restart docker

参考链接：

需要登录阿里云帐户

https://cr.console.aliyun.com/cn-shanghai/instances/mirrors

安装portainer

portainer一款轻量级的docker UI管理工具，真香！

1
2
3
4
5

sudo docker pull portainer/portainer-ce:alpine
sudo docker volume create myportainer_data
sudo docker run -d -p 127.0.0.1:9000:9000 --name=myportainer --restart=always \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v myportainer_data:/data portainer/portainer-ce:alpine

忘记密码重置方法

1
2
3

sudo docker stop myportainer
sudo docker run --rm -v myportainer_data:/data portainer/helper-reset-password
sudo docker start myportainer

参考链接：

https://docs.docker.com/network/iptables/

https://hub.docker.com/r/portainer/portainer-ce
